# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean and modern user interface.

---

## 🚀 Features

* ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
* 🔢 Supports decimal and double zero (00) input
* ⌫ Delete last digit
* 🧹 All Clear (AC) functionality
* 📊 Percentage calculation
* ⌨️ Keyboard support
* 🎨 Modern UI with smooth animations

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
